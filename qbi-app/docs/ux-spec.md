# ТЗ: оптимизация главного экрана QBI-приложения

## 1. Проблема

Сейчас на главном экране одним длинным скроллом лежит всё сразу: герой-текст с буллетами, баннер, 3 карточки "что делает", 4 карточки экспертизы, аккордеон услуг, сетка рубрик, блок статистики. Это 8+ смысловых блоков подряд — человек либо теряется, либо просто долистывает не читая.

**Принцип решения:** на главном экране остаётся только вход в навигацию и один яркий оффер. Всё содержательное (объяснения, экспертиза, услуги, статистика) переносится "за клик" — либо в поп-ап при первом входе, либо в разделы нижней навигации.

Ориентир по паттерну — референс с кофейным приложением: верх — поиск, крупный цветной баннер, сетка рубрик с картинками, низ — фиксированная навигация с иконками. Переносим этот принцип на финансовый контент.

## 2. Структура главного экрана (сверху вниз, без длинного скролла)

1. **Шапка**: логотип QBI + иконка поиска (не строка на весь экран, а кнопка/поле, разворачивается по тапу)
2. **Баннер** — один экран высотой ~25-30% от вьюпорта, яркий цветной фон (не тёмная панель, а акцентный градиент — как промо-баннер в референсе). Ротирующийся контент: обещание продукта / совет дня / приглашение на диагностику. Один баннер, не список.
3. **Сетка рубрик с иконками** — основная навигация по контенту, крупные плитки с картинкой/иконкой + числом (60 цитат, 20 терминов и т.д.), без текста-простыни рядом.
4. Дальше — **пусто**. Экран заканчивается после сетки рубрик, скролла почти нет.

Все остальные текущие блоки (обещания-буллеты, "что делает приложение", 4 карточки экспертизы, статистика Максата) переносятся внутрь соответствующих разделов, не дублируются на главной.

## 3. Нижняя навигация (фиксированная, 5 вкладок)

| Иконка | Раздел | Что внутри |
|---|---|---|
| 🏠 | Главная | баннер + сетка рубрик (см. п.2) |
| 💬 | ИИ-консультация | диагностика (текущий чат-экран) |
| 📚 | Знания | цитаты / глоссарий / бухгалтер vs финансист / находки |
| 🧩 | Услуги | 4 продукта QBI + 2-дневный интенсив (текущий аккордеон) |
| ⭐ | Профиль/О нас | статистика Максата, экспертиза QBI по 4 направлениям, контакты |

Нижняя навигация видна всегда (кроме, возможно, полноэкранного режима чата и карточек-свайпов, где вместо неё — кнопка "назад").

## 4. Поп-ап онбординга (первый запуск)

- Показывается **один раз** при первом открытии приложения (флаг в рамках сессии/устройства — не блокирует повторные визиты).
- 2-3 экрана-свайпа: что это, как работать, что получите на выходе — тот текст, что сейчас "обещания" списком в hero, переезжает сюда.
- Кнопка "Начать" закрывает поп-ап навсегда — дальше он **не занимает место** на главном экране, доступен по желанию через раздел "Профиль/О нас" → "Как это работает" (маленькая ссылка).

## 5. Поиск (верхняя иконка)

- По тапу разворачивается строка поиска.
- Ищет по всей "Библиотеке знаний" сразу: глоссарий (термин → определение), цитаты, находки, сравнения — единый поиск, результаты — те же карточки, что и в разделе "Знания".
- Это прямой ответ на сценарий "что это? [ввёл термин] → увидел объяснение", не обязательно идти в раздел через рубрику.

## 6. Правило сворачивания контента

Общий принцип для дизайнера: **ничего объёмного не лежит открытым по умолчанию**, если это не единственная цель экрана.
- Списки правил/буллетов → в поп-ап или в подраздел, не на главную
- Разворачивающиеся описания услуг → остаются аккордеоном, но **внутри вкладки "Услуги"**, не на главной
- Статистика и "о QBI" → отдельная вкладка/шапка профиля, не на главной

## 7. Что переносится и куда (сводная таблица)

| Сейчас на главной | Переносится в |
|---|---|
| Hero-текст с 4 буллетами-обещаниями | Поп-ап онбординга (п.4) |
| 3 карточки "что делает приложение" | Поп-ап онбординга (п.4) |
| Баннер "совет дня" | Остаётся, но становится единственным баннером экрана (п.2) |
| 4 карточки "экспертиза QBI" | Вкладка "Профиль/О нас" |
| Аккордеон "Услуги" (5 карточек) | Вкладка "Услуги" |
| Сетка рубрик библиотеки | Остаётся на главной как основная навигация (п.2-3) |
| Статистика (19 лет / 110+ / 1,5 трлн) | Вкладка "Профиль/О нас" |

## 9. Финальные решения (зафиксировано)
- **Иконки рубрик на главной** — дружелюбные иллюстрации/эмодзи в цветных бейджах-кружках (как в примере с кофе). «Приборная панель» (моноширинные цифры, уголки, сетка) остаётся только **внутри** разделов — в чате, в карточках библиотеки, в счётчиках.
- **Нижняя навигация — 4 вкладки** (без «Профиль/О нас»):
  1. 🏠 Главная — баннер + сетка рубрик
  2. 💬 ИИ-консультация
  3. 📚 Знания — цитаты / глоссарий / бухгалтер vs финансист / находки + карточки «экспертиза QBI»
  4. 🧩 Услуги — 4 продукта + интенсив, статистика Максата (19 лет / 110+ / 1,5 трлн) — короткой строкой сверху, как аргумент доверия перед ценами.
